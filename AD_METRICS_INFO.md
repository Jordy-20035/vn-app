# Информация о метриках рекламы

## Текущее состояние

На данный момент в приложении **нет встроенных инструментов** для подсчёта метрик рекламы (клики, просмотры). 

## Варианты решения

### 1. **Telegram Ads API** (Рекомендуется)
Telegram предоставляет встроенный API для рекламы в Mini Apps:
- Автоматический подсчёт показов и кликов
- Интеграция через `WebApp.showAd()` или `WebApp.openTelegramLink()`
- Метрики доступны в Telegram Ads Manager

**Плюсы:**
- ✅ Встроенная аналитика
- ✅ Не требует дополнительных сервисов
- ✅ Работает напрямую с Telegram Stars

### 2. **Яндекс.Директ**
Можно подключить Яндекс.Директ к баннерам:
- Использовать Яндекс.Метрику для отслеживания
- Добавить UTM-метки к ссылкам
- Настроить цели в Метрике

**Плюсы:**
- ✅ Гибкая настройка кампаний
- ✅ Детальная аналитика
- ✅ A/B тестирование

**Минусы:**
- ⚠️ Требует настройки Яндекс.Метрики
- ⚠️ Нужно добавлять код отслеживания

### 3. **Собственная аналитика**
Можно добавить простой трекинг:
- Отправка событий на ваш backend при показе/клике рекламы
- Хранение метрик в базе данных
- Собственный дашборд

**Плюсы:**
- ✅ Полный контроль
- ✅ Кастомные метрики

**Минусы:**
- ⚠️ Требует разработки backend
- ⚠️ Нужна инфраструктура для хранения данных

## Рекомендация

Для начала использовать **Telegram Ads API** — это самый простой и эффективный способ для Mini Apps. Если понадобится более детальная аналитика, можно добавить Яндекс.Метрику как дополнительный инструмент.

## Что нужно для интеграции

1. **Telegram Ads:**
   - Настроить рекламный кабинет в Telegram
   - Использовать `WebApp.showAd()` в компоненте `AdScreen.jsx`

2. **Яндекс.Директ (опционально):**
   - Создать счётчик Яндекс.Метрики
   - Добавить код отслеживания в `index.html`
   - Настроить цели для кликов по рекламе

Могу помочь с интеграцией любого из этих вариантов!
