# UI Assets Required

This directory should contain the following UI images for the game interface:

## Currency Counters
- `energy-counter-bg.png` - Background plate for energy counter
- `rubies-counter-bg.png` - Background plate for rubies/diamonds counter
- `energy-icon.png` - Lightning bolt icon for energy
- `rubies-icon.png` - Diamond/ruby icon for currency

## Ad Bonus Button
- `ad-bonus-bg.png` - Background plate for ad bonus button (with "Реклама +2" text area)

## Shop Assets
- `shop-beginner-chest.png` - Chest image for beginner pack
- `shop-best-chest.png` - Large chest image for best value pack
- `shop-medium-chest.png` - Medium chest image
- `shop-small-chest.png` - Small chest image
- `shop-energy-bolt.png` - Lightning bolt image for energy packs

## Notes
- All images should be optimized for web (PNG with transparency where needed)
- Recommended size: 80x80px for icons, larger for chest images
- The counter backgrounds should have space for numbers/text overlay
